# 📚 Documentation & Livrables

Ce répertoire centralise tous les documents non-techniques liés à la gestion du projet et aux livrables attendus pour la SAE 401.

## 📑 Contenu
* **Cahier des charges :** Les exigences initiales du client.
* **Matrice de Suivi QA (Excel) :** Fichier répertoriant nos tests, croisant l'impact et la difficulté d'exploitation pour calculer le score ANSSI (Mineur, Important, Majeur, Critique).
* **Rapport Technique Final :** Le document PDF (livrable) détaillant la conception de notre plateforme SaaS et de notre système de reporting.
* **Support de Présentation :** Le diaporama pour la soutenance orale.
